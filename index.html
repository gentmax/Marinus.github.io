What is Marinus.css?
It is a lightweight css framework designed to help web developers be more productive and eficient
What is the size of this framework?
The size of the framework is only 5kb making it one of the most lightweight frameworks on the entire internet

Marinus is based on flexbox layout and is fully responsive in any screen size

You can download it here directly from our page or get the source code from github
